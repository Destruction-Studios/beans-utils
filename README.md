# Description
Adds many multiplayer server features.
Also adds many QOL features on the client.

### THIS MOD IS STILL IN BETA ###
**VERSIONS BELOW 0.4.0 REQUIRE [FZZY-CONFIG](https://modrinth.com/mod/fzzy-config)**

## Current Features

**Server Features**
- Combat Tagging
- TPA commands
- Toggling of dimensional portals
- Runtime hash/resource pack updating

**Client Features**
- Prevents leaving when in combat
- Removes to need to reload resource packs if one of same hash is already loaded

## Planned Features

**Server**
- Respawning Pets

**Client**
- Last joined display on servers

and more I cant remember

All of these settings can be toggled and customized with the built in config.
